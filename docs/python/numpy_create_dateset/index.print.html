Creating dateset by Numpy

データの生成

等差数列の生成

np.arange(start, stop, step)

要素数を指定した等差数列の生成

np.linspace(start, stop, num, endpoint=True)
  • endpoint: stopを数列に含むか否か（デフォルトはTrue）

同じ値で初期化した配列の生成

  • 0で初期化

    np.zeros(shape)

    np.zeros_like(arr)

  • 1で初期化

    np.ones(shape)

    np.ones_like(arr)

  • 任意の値 (fill_value) で初期化

    np.full(shape, fill_value)

    np.full_like(arr, fill_value)

  • 空の配列を生成

    np.empty(shape)

    np.empty_like(arr)

単位テンソルの生成

np.eye(shape)
